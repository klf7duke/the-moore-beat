HOW TO UPDATE YOUR SITE
========================

You only ever edit ONE file: events.js

To add an event, special, or business, open events.js, copy an existing
{ ... } block, paste it, and change the words between the quote marks.
Save the file. That's it — the website updates automatically.

PAST EVENTS DISAPPEAR ON THEIR OWN. Once an event's date has passed, it
stops showing on the site, so you don't have to delete old ones right away.

WANT CLAUDE TO DO IT FOR YOU?
Just ask, for example:
  "Add this weekend's events in Moore County to my site."
  "Pull the June events from the Pinehurst village calendar and add them."
  "Add a special: 20% off at Joe's Diner through Friday."
Claude will edit events.js for you.

THE FILES
  index.html  ........ the website itself (you rarely touch this)
  events.js   ........ all events, specials, and businesses (edit this)
  README ............. this file

TO SEE THE SITE: double-click index.html to open it in your web browser.
TO PUT IT ONLINE: see the step-by-step guide that came with this folder.
